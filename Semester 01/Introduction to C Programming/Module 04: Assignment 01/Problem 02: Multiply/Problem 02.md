Problem Statement
You will be given two integers A and B. You need to give the output of their multiplication.

Input Format
- Input will contain A and B

Constraints
- -10^9 <= A,B <= 10^9

Output Format
- Output the multiplication

Sample Input 0
<pre>10 50</pre>
Sample Output 0
<pre>500</pre>
Sample Input 1
<pre>10000000 10000000</pre>
Sample Output 1
<pre>100000000000000</pre>
Sample Input 2
<pre>-100 62</pre>
Sample Output 2
<pre>-6200</pre>
