Problem Statement

Take two integers A and B as input and output their sum.

Input Format
- You will be given A and B separated by a space.

Constraints
- -10^9 <= A,B <= 10^9

Output Format
- Output their summation

Sample Input 0
<pre>2 3</pre>
Sample Output 0
<pre>5</pre>
Sample Input 1
<pre>-10 5</pre>
Sample Output 1
<pre>-5</pre>
